高级表单部件

数字

用于数字的小部件是用 <input>元素创建的，它的 type 属性设置为 number。这个控件看起来像一个文本域，但是只允许浮点数，并且通常提供一些按钮来增加或减少小部件的值。

也可以：

  • 通过设置 min 和 max 属性来约束该值。
  • 通过设置 step 属性来指定增加和减少按钮更改小部件的值的数量。

滑块

另一种选择数字的方法是使用滑块。从视觉上讲，滑块比文本字段更不准确，因此它们被用来选择一个确切值并不重要的数字。滑块是通过把 <input> 元素的 type 属性值设置为 range 来创建的。正确配置滑块是很重要的；为了达到这个目的，我们强烈建议您设置 min、max 和 step 属性。

日期时间选择器

对于 web 开发人员来说，收集日期和时间值一直是一场噩梦。HTML5 通过提供一种特殊的控制来处理这种特殊的数据，从而带来了一些增强。使用 <input> 元素和一个 type 属性的适当的值来创建日期和时间控制，这取决于是否希望收集日期、时间或两者。

所有日期和时间控制都可以使用 min 和 max 属性来约束。

拾色器

颜色总是有点难处理，有很多方式来表达它们: RGB 值(十进制或十六进制)、HSL 值、关键字等等。颜色小部件允许用户在文本和可视的方式中选择颜色，颜色小部件是使用 <input> 元素创建的，它的 type 属性设置为值 color。
